This package provides base classes that provide XML setData/getData support to implementations of the {@link littleware.asset.Asset} interface. These classes are intended for implementors of new Asset types.

An Asset subtype may use the {@link littleware.asset.xml.XmlGetter} and {@link littleware.asset.xml.XmlSetter} annotations in conjunction with the {@link littleware.asset.xml.XmlDataAsset} abstract base class to manage the automatic marshalling of subtype specific data to and from the {@link littleware.asset.Asset} getData/setData XML string.
