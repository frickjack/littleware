littleId service

Easily OpenId enable your application's authentication process with littleId. The littleId service acts as an openId assistant that coordinates a web application's login process with an openId consumer hosted locally or at services.frickjack.com to allow a client to authenticate with Google or Yahoo credentials.

Take a look at the demo to get started. Next, download the littleware SDK, browse the online scaladoc and ydoc (javascript) API documentation, or access the source code repository (including the PHP client). Litleware is open-source under the LGPL license.
